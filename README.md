---

## Задача проекта

Разработать консольное приложение для автоматизации учёта товаров на складе.

Программа поддерживает:

- добавление товаров;
- редактирование записей;
- удаление товаров;
- поиск и фильтрацию;
- сортировку;
- аналитику;
- сохранение и загрузку данных из TXT-файла.

---

## Технические требования

- Язык: C++17
- STL: vector, algorithm
- Работа с файлами: fstream
- Архитектура: модульная
- Формат хранения: TXT
